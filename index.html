HTML-sivun rakenne

Perus HTML-sivu koostuu seuraavista osista:

<html> - koko sivu
<head> - metatiedot
<body> - näkyvä sisältö

<HTML>

Määrittelee HTML-dokumentin alun ja lopun.

<HEAD>

Sisältää metatiedot, kuten sivun otsikon ja linkit tyyleihin.

<BODY>

Sisältää kaiken sivulla näkyvän sisällön.

<P>

Kappaletekstiä varten.

<IMG>

Kuvan lisääminen sivulle.

<img src="kuva.jpg" alt="kuva">

<A>

Linkkien luominen.

<a href="https://example.com">Linkki</a>

<BR>

Rivinvaihto.

<H1>

Pääotsikko.

CSS-muotoilu

CSS:llä määritellään sivun ulkoasu. Tyylejä voidaan lisätä:

  • Inline-tyylinä (style="...")
  • Head-osassa <style>-tagissa
  • Erillisessä .css-tiedostossa
body { background-color: lightgray; }

Bootstrap

Bootstrap on valmis CSS-kirjasto, joka helpottaa sivujen rakentamista.

  • Valmiit komponentit (navbar, card, button)
  • Responsiivinen rakenne
  • Nopea kehitys
